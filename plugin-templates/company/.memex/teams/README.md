# Teams

One folder per team. Each folder must contain a `README.md` with the charter:

- **Mission** — what the team is responsible for
- **Members** — links to people pages
- **Cadence** — meetings, async rituals
- **Active OKRs** — link into [OKRs](../okrs/)
