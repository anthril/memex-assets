# People

One page per teammate. Filename convention: `firstname-lastname.md`. Keep contact details and expertise current — the hooks block writes that aren't kebab-case.

Each page should include:

- **Role** — what they do day-to-day
- **Reports to** — link to manager
- **Expertise** — areas they own / can be consulted on
- **Contact** — email, Slack, time zone
