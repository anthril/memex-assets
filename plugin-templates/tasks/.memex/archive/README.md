# Archive

Closed tasks live here. Archive policy:

- Move a task into `archive/` one week after it hits `status: done`.
- Keep its `date:` field intact so the Timeline view still surfaces the
  history when filtered to "All".
- Don't change `status:` after archiving — the Kanban "Done" column reads from
  `tasks/` only.
