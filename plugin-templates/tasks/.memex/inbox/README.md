# Inbox

Quick-capture zone for un-triaged items. Drop one `.md` per idea / bug report /
follow-up here. Promote into `tasks/` (or `epics/` / `milestones/`) once you've
fleshed out the frontmatter and decided where it belongs.

Inbox items appear in the **Backlog** Kanban column when they carry
`status: backlog`.
