# Open questions

One file per question. Resolved questions move to `.resolved/`.

Use `/memex:open-q <title>` to file a new one.

## Required: link the related files

Every open question **must** include a `## Related files` section with markdown links to every page or file the question touches: what prompted it, what depends on the answer, what will need updating once it's resolved. Use `[label](relative/path)`; bare paths do not count.

If you cannot identify any related file, the question probably belongs inline on the most-affected page rather than as a cross-cutting question.
