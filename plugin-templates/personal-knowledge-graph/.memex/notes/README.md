# Atomic notes

One idea per note. Filename convention: `YYYYMMDDHHMM-kebab-case-slug.md`. The New Page command auto-prefixes the timestamp.

If a note grows past ~500 words OR contains multiple distinct ideas, split it. Wikilink to the children and back.
