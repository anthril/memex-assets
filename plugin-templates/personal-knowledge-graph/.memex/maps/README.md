# Maps of content (MOCs)

Hand-curated overview pages that link a cluster of related notes. The top-level [index.md](../index.md) is the entry MOC; create more as your graph grows.

Each MOC opens with one paragraph that frames the topic, then sections of wikilinks grouped by sub-theme.
